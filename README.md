# Exchange 
**Exchange for test different currencies. Made with Html, Css and JavaScript. Using jQuery, Bootstrap for styles and Cypress for testing.**

### Installation:
- download or clone repo
- run: ***npm install***

#### for run tests:
- Install http-server

- run: ***npm run start***

- run: ***npm run cypress:open***
